# Millicent Muema – portfolio
My portfolio showcases my projects, testimonials, and professional branding.  
Built with **HTML, CSS, JavaScript, Bootstrap**, and integrated with the **WordPress REST API** for dynamic testimonials.

---

## 🚀 Features
- **Responsive design** – seamless across desktop, tablet, and mobile.
- **Dynamic Testimonials** – recruiters and peers can submit testimonials directly via the portfolio form.
- **WordPress Integration** – testimonials are stored and displayed using the WordPress REST API.
- **Project Gallery** – interactive cards highlighting completed and upcoming projects.
- **Clean Branding** – consistent layout and design for professional presentation.

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Bootstrap  
- **Backend:** WordPress REST API  
- **Version Control:** Git & GitHub  
- **Hosting:** GitHub Pages / Netlify / (your chosen host)
